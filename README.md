# Sagar Mewada — Portfolio

A personal portfolio website built with React, showcasing my projects, skills, and background in AI, Data Science, and full-stack development.

🔗 **Live Demo:** _add your deployed link here (e.g. Vercel/Netlify URL)_

## ✨ Features

- Smooth scroll experience powered by [Lenis](https://lenis.darkroom.engineering/)
- Scroll-triggered reveal animations and hover micro-interactions via [Framer Motion](https://www.framer.com/motion/)
- Typewriter-style rotating role tagline in the hero section
- Interactive skill chips with hover tooltips describing hands-on experience
- Education section with hover-to-expand cards showing scores and highlights
- Fully responsive, built with Tailwind CSS v4
- No external icon library — lightweight inline SVG icons

## 🛠️ Tech Stack

- **Frontend:** React (Vite)
- **Styling:** Tailwind CSS v4
- **Animation:** Framer Motion, Lenis
- **Icons:** Custom inline SVGs

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

```bash
git clone https://github.com/sagar9876-bit/your-repo-name.git
cd your-repo-name
npm install
```

### Run locally

```bash
npm run dev
```

Visit `http://localhost:5173` to view it in the browser.

### Build for production

```bash
npm run build
```

Output is generated in the `dist/` folder.

## 📁 Project Structure

```
src/
├── App.jsx        # Main portfolio component (all sections)
├── index.css      # Tailwind import
└── main.jsx        # React entry point
```

## ✏️ Customization

All editable content — name, roles, skills, projects, and education — lives in clearly marked constant objects (`PROFILE`, `SKILLS`, `PROJECTS`, `EDUCATION`) near the top of `App.jsx`. Update these to personalize the site without touching layout code.
